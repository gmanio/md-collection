# md-collection

이 페이지는 저장소의 Markdown 문서를 GitHub Pages로 렌더링하기 위한 기본 홈입니다.

## 사용 방법

- 루트(또는 하위 디렉터리)에 `.md` 파일을 추가합니다.
- `main` 브랜치에 변경사항이 push되면 GitHub Actions가 자동으로 Pages를 배포합니다.
- 배포 후 `https://<github-username>.github.io/<repository-name>/`에서 확인할 수 있습니다.
